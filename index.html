01. INTRO

강화학습이란? (당근과 채찍)

인공지능을 학습시키는 방법 중 하나인 '강화학습(Reinforcement Learning)'은 강아지에게 '앉아!'를 가르치는 것과 똑같습니다.

  • 잘한 행동(앉기)을 하면 간식(보상)을 줍니다. -> 그 행동을 더 자주 하게 됨
  • 나쁜 행동(물기)을 하면 혼(패널티)을 냅니다. -> 그 행동을 안 하게 됨

결과적으로 AI는 "어떻게 행동해야 보상을 가장 많이 받을까?"를 스스로 깨닫는 자신만의 룰(정책, Policy)을 만들게 됩니다.

02. THE PROBLEM

기존 방식의 문제: 너무 큰 깨달음의 부작용

자전거 타기를 배울 때를 상상해 보세요.

자전거를 타다가 몸이 오른쪽으로 기울었을 때, "아! 왼쪽으로 핸들을 꺾어야겠다!"라는 깨달음을 얻습니다.

하지만 초보자가 이 깨달음에 너무 흥분해서 핸들을 왼쪽으로 '확!' 꺾어버리면 어떻게 될까요?
반대편으로 심하게 넘어지면서 오히려 자전거 타는 법을 완전히 까먹거나 다치게 됩니다. (이를 AI 용어로는 학습 붕괴, Catastrophic Forgetting라고 합니다)

기존 방식 (Policy Gradient)

좋은 결과를 얻으면 핸들을 확 꺾어버려서 종종 학습이 완전히 망가짐.

PPO의 아이디어

"아무리 좋은 깨달음을 얻어도, 한 번에 조금씩만 핸들을 꺾게 만들자!"

03. MATHEMATICAL CORE

수식을 일상어로 번역하기: 20% 안전벨트

PPO의 논문에 나오는 무서운 수식은 사실 아주 단순한 "안전벨트 규칙"일 뿐입니다.

min( r(θ) × A , clip(r(θ), 0.8, 1.2) × A )

💡 한 줄 요약:

"어떤 행동으로 칭찬(A)을 받았을 때, 그 행동을 할 확률(r)을 높이되, 이전보다 최대 1.2배(20%)까지만 높이도록 잘라버려라(Clip)!"

r(θ) 행동 변화량

과거의 나에 비해 새로운 내가 이 행동을 얼마나 더 자주 하려고 하는가?

A 어드밴티지 (칭찬)

이 행동이 평소보다 얼마나 더 좋은 결과를 가져왔는가?

Clip (0.8 ~ 1.2)

핵심 안전장치. 아무리 칭찬을 많이 받아도 한 번에 행동 패턴을 20% 넘게 바꾸지 못하게 브레이크를 겁니다.

05. THE CODE

현업에서는 코드로 어떻게 짤까?

앞서 배운 '20% 안전벨트(Clipping)' 원리가 실제 PyTorch 코드로는 단 한 줄로 구현됩니다.

ppo_core.py 
def ppo_loss(ratios, advantages):
    # 1. 제한 없이 깨달은 대로 반영해보기 (일반적인 방식)
    surr1 = ratios * advantages
    
    # 2. 안전벨트 작동! 아무리 변해도 0.8 ~ 1.2 사이로 잘라버리기 (Clip)
    clipped_ratios = torch.clamp(ratios, min=0.8, max=1.2)
    surr2 = clipped_ratios * advantages
    
    # 3. 둘 중 '더 보수적인(작은)' 값을 최종으로 선택 (급발진 방지)
    final_loss = -torch.min(surr1, surr2).mean()
    
    return final_loss
💡 바로 이 torch.clamp( ..., 0.8, 1.2 ) 코드가 인공지능이 무너지지 않고 꾸준히 똑똑해지게 만드는 마법의 함수입니다!
